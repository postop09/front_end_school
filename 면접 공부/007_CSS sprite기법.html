CSS Sprite

CSS Sprite기법에 대한 정의와 장단점에 대해 설명해주세요.

CSS image sprite라고도 불리는데, 말 그대로 이미지를 합치는 기법입니다. 조금 더 자세히는 여러개의 이미지를 하나의 큰 도화지 안에 넣어서 관리할 수 있는 기법입니다. 그리고 그 도화지를 이동하며 원하는 이미지만 보이도록 하는 것입니다. 이렇게 함으로써 문서는 많은 이미지를 불러올 부담이 줄어들게 되고, 이것은 곧 오고가는 요청이 줄어들게 됩니다. 즉, 랜더링 속도가 빨라진다는 장점이 있을 수 있습니다. 하지만 너무 많은 이미지를 합친다면 사이즈가 커지면서 불러오는데 시간이 오래 걸리는 단점이 있으며, 그러므로 주제별로 나누어서 사용하는 것이 좋은 방법이 되겠습니다. 또한 사용하는 이미지가 변경되면 위치값을 재조정해야 하기 때문에 잦은 수정이 있는 이미지는 스트라이프를 하지않는 것이 좋습니다.
